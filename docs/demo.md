# Demo mode

Stage 1 includes a polished browser preview of the Telegram Mini App. It demonstrates:

- responsive mobile-first layout;
- UA/EN localization foundation;
- light/dark Telegram theme integration;
- active search summary placeholder;
- system health states;
- apartment card and future Match Score presentation;
- bottom navigation and haptic hooks;
- safe behavior outside Telegram.

The values shown in the preview are explicitly presentational and are not persisted as listings.

Stage 3 will replace this presentation-only data with at least 150 synthetic listings, demo sources, duplicates, price history and a reproducible seed command.
